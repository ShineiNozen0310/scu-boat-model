# 船模项目淘宝采购清单

更新日期: 2026-03-11

## 1. 适用范围

这份清单按当前项目方案整理:

- 船体: 1/30 飞鱼号板件
- 控制链路: ELRS 2.4GHz 成品遥控 + 船端 CRSF
- 主控: STM32F103
- 推进: 单有刷电机
- 操舵: 单方向舵机
- 甲板机构: 双轴水炮/喷口

说明:

- 价格是淘宝常见区间的工程估算, 用于控制预算, 不是某一家店的实时成交价。
- 这份清单偏“先做出稳定可调试版本”, 不追求极限性能。
- 如果你已经买过部分元件, 直接从“必买”和“不要再买”两节对照删减即可。

## 2. 采购结论

最推荐的低成本主方案:

- 遥控器: RadioMaster Pocket ELRS 2.4G
- 接收机: RadioMaster RP1 V2
- 主控板: STM32F103C8T6 最小系统板
- 动力: 280 有刷电机 + 20A~30A 有刷电调 + 2mm 轴系 + 30~35mm 三叶桨
- 方向舵: SG92R
- 水炮: SG92R + SG90 + 二自由度云台 + 75mm 金属喷水嘴 + S05 水枪脉冲齿轮箱
- 电源: 2S 电池 + 5V 3A UBEC

预算估算:

- 不含遥控器: 约 320~560 元
- 含遥控器: 约 760~1080 元

## 5. 不建议再买的东西

这些不是完全不能用, 而是对你现在这条技术路线不划算:

| 元件 | 结论 | 原因 |
| --- | --- | --- |
| E32-900M20S | 不建议再买 | 现在主链路已经改成 ELRS + CRSF, E32 是 SPI LoRa 模块, 不再是主遥控方案 |
| MG996R | 不建议作为这条小船的默认舵机 | 55g 太重, 电流大, 体积也大; 除非你的甲板机构特别重 |
| 单节 18650 3.7V | 不建议做全船主电 | 舵机、接收机、主控、电调供电不顺, 后面会被迫加升压和更多补丁 |
| 130 马达 + 49mm 六叶桨 | 不建议做主推进 | 小马达拉大桨, 电流和效率都不漂亮, 调起来会比较痛苦 |
| 95mm 吸水炮 | 不建议作为默认选型 | 对 1/30 小船偏大, 更容易占甲板空间和压重心 |

## 6. 性价比优化建议

### 优先省钱的地方

1. 遥控器选 Pocket, 不要一开始上更大的手台。
2. 接收机先用 RP1 V2, 不急着上双天线。
3. 主控继续用 STM32F103 最小系统板, 不要为了“看起来高级”换更贵 MCU。
4. 推进先用有刷电机 + 有刷 ESC, 别一开始切无刷。
5. 方向舵和水炮水平轴都用 SG92R, 俯仰轴用 SG90, 这是当前性价比最平衡的组合。

### 不该省的地方

1. UBEC 不要省掉。
2. 主电保险和电源开关不要省掉。
3. 轴系、联轴器、方向舵总成尽量买同一家配套件, 少走弯路。
4. 接收机天线不要图小巧选全内置陶瓷款, 小船更适合外置 T 天线。

## 10. 关键选型依据

- ELRS 官方文档对新用户默认更推荐 2.4GHz, 900MHz 更偏长距离或高噪声环境。
- Pocket 官方页确认其为 2.4GHz ELRS 版本、EdgeTX、双 18650、霍尔摇杆。
- RP1 官方页确认其工作电压为 5V, 总线接口为 CRSF, 且使用外置 65mm T 天线。
- SG92R 官方规格比 SG90 扭矩更大, 但成本仍低, 更适合作方向舵和水平旋转轴。
- MG996R 官方规格显示其重量和电流都明显更大, 对这条小船默认方案性价比不高。
- E32 官方页显示它是 850~931MHz 的 SX1276 LoRa SPI 模块, 不适合当前 ELRS + CRSF 主链路。
