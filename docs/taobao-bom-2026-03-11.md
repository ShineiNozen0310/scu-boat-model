# 船模项目淘宝采购清单

更新日期: 2026-03-13

## 1. 适用范围

这份清单按当前项目方案整理:

- 船体: 1/30 飞鱼号板件
- 控制链路: 预算优先时走 `DIY 发射端 + E220 UART LoRa + boat_radio_protocol`; 预算放宽后可升级 `ELRS + CRSF`
- 主控: STM32F103
- 推进: 单有刷电机
- 操舵: 单方向舵机
- 甲板机构: 双轴水炮/喷口

说明:

- 这次调整的目标是把“整船必买项”压到 800 元以内。
- 目标同时包含三件事: 稳定开船、一定抗干扰能力、支持手动打靶控制。
- 这里的预算按工程估算给出, 用于压总价, 不是某一家店的实时成交价。
- 预算方案能做, 但它不是仓库里抗干扰最强的方案; 真正更稳的仍然是成熟 `ELRS -> CRSF`。
- `900MHz` 类模块的频段和发射功率是否合规, 需要按你所在地法规自行确认。

## 2. 采购结论

### 当前推荐主方案

如果你现在最硬的约束是“整船预算控制在 800 元以内”, 推荐改成:

- 发射端主控: ESP32 开发板
- 发射端输入: 2 个双轴摇杆 + 若干拨杆/按键
- 发射端无线模块: EBYTE `E220-900T22D`
- 接收端无线模块: EBYTE `E220-900T22D`
- 船端接入: STM32 UART 收透明传输数据, 然后走 `MainApp_OnRadioPacket(...)`
- 推进: 280 有刷电机 + 20A 级有刷 ESC
- 舵机: 方向舵 `SG92R`, 水炮水平 `SG92R`, 水炮俯仰 `SG90`

预算估算:

- 预算型 DIY 遥控链路: 约 109~173 元
- 含整船但不含可选件: 约 464~770 元

### 为什么不是 ELRS

不是因为 ELRS 不好, 而是因为在 2026-03-13 这个时间点:

- `RP1 V2` 官方价仍是 `$18.99`
- 成熟 ELRS 发射模块常见店价仍在 `$25~40+`
- 如果加上你还要手搓的发射端结构、电源和摇杆, 很容易把总预算顶出 800

所以当前这份 BOM 先按“预算内可落地”来定。

### 如果后面预算放宽

更稳的升级方案仍然是:

- 发射端: 成品 ELRS 手台或 DIY 外设 + ELRS 发射模块
- 接收端: `RP1 V2` 或更高一级的真分集接收机
- 船端: `CRSF -> STM32`

## 6. 压预算但不牺牲核心体验的做法

### 可以省的地方

1. 发射端不先上 OLED、霍尔摇杆和 3D 打印外壳。
2. 无线链路先用 `ESP32 + E220 + 基础摇杆`, 不买成品手台。
3. 水炮俯仰轴用 `SG90`, 不强上更贵舵机。
4. 充电器先用入门 `B3` 级平衡充。
5. 主电池先用 `1500~2200mAh` 2S, 不追求大容量。

### 不该省的地方

1. 方向舵和水炮水平轴不要降到杂牌弱舵机。
2. 船端 UBEC 不要省掉。
3. 主电保险和电源开关不要省掉。
4. 接收端无线模块安装位置要远离电机、ESC 和主电池。
5. 天线不要乱配, 模块频段和天线接口要匹配。

## 8. 和当前仓库代码的关系

预算型无线链路建议直接对接:

- 船端入口: `firmware/app/main_app.c` 里的 `MainApp_OnRadioPacket(...)`
- 协议封装: `firmware/app/drivers/boat_radio_protocol.*`

这样做的好处是:

- 不需要把预算方案硬塞进 `CRSF`
- 发射端 MCU 只要打包 8 字节控制帧即可
- 当前命令结构已经包含 `油门/方向/炮塔水平/炮塔俯仰/水炮开关`
- 后面如果预算放宽, 再切回 `ELRS -> CRSF` 也不影响上层控制逻辑

## 11. 关键选型依据

- `E220-900T22D` 官方页显示它是 UART 透明传输 LoRa 模块, 基于 `LLCC68`, 发射功率 `22dBm`, 典型最远 `5km` 级别。
- EBYTE 官方商城当前给出的 `E220-900T22D` 单价是 `$4.19`; 作为预算遥控链路核心, 它比 ELRS 模块明显更容易压住总价。
- `RP1 V2` 官方页当前标价 `$18.99`, 接收机本身就会显著抬高 800 内方案的预算压力。
- `SG92R` 在常见零售渠道的单价仍然处于微型舵机可接受区间, 用在主方向和水炮水平轴比 `SG90` 更稳。
- `B3` 类 2S/3S 平衡充当前常见店价仍在低价位区间, 适合作为预算版首套充电器。
