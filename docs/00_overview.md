# 00 概览

## 项目目标

这个项目当前以“红外遥控小船基础功能可复现、结构可扩展”为目标，先把最核心的链路稳定下来：

- 红外遥控接收
- 电机前进 / 后退 / 停止
- 舵机左右转向
- LED 状态反馈
- 基础失联保护和急停接口

## 原始工程存在的问题

根据 `B站up 红外遥控小船源代码.rar` 中的 Keil 工程，原始代码可以工作，但结构上有几个明显问题：

- `TIM2_IRQHandler()` 里同时做输入捕获、协议解析和状态修改
- `Motor_Speed`、`Motor_Dir`、`Servo_Angel` 等状态是跨模块全局变量
- OLED、LED、马达和红外回复逻辑耦合在主循环与中断之间
- 参数分散在多个源文件里，后期调参容易漏改

## 当前仓库的整理原则

- 生成代码和手写逻辑分离
- 中断只采集数据，不直接改业务状态
- 控制逻辑、驱动抽象、安全策略拆开
- 文档、硬件、机械、测试资料各自归档

## 当前目录的职责

- `docs/`：方案和过程文档
- `firmware/cubemx/`：CubeMX/Keil 工程
- `firmware/app/`：手写固件逻辑
- `hardware/`：接线图、BOM、原理图
- `mechanical/`：船体和装配资料
- `test/`：台架与下水测试记录

## 近期可执行事项

1. 把你现在能跑的板级工程放进 `firmware/cubemx/`。
2. 用 `docs/03_electronics.md` 的引脚表核对一次接线。
3. 在 `test/bench/` 先完成“红外解码 + 电机 + 舵机”三项台架验证。
4. 下水前先补充 `mechanical/assembly_notes.md` 里的防水和重心检查。
