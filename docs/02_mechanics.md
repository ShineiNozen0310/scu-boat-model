# 02 机械设计

## 当前建议的机械目标

因为现在仓库里还没有实际船体图纸，先把后续设计时最容易出问题的点固定下来。

## 船体布置建议

- 电池尽量靠近船体纵向中心，减少前后俯仰
- 电机、联轴器、推进轴保持尽量同轴，避免高转速抖动
- 舵机和拉杆尽量走直线，减少卡滞
- 接收头不要被船壳或装饰件遮挡
- 主控板和驱动板固定时预留检修空间，不要直接热熔胶封死

## 防水和结构刚性

- 船体上盖建议做可重复拆装的密封结构
- 电机轴、舵杆出水位置建议做单独密封
- 电子仓底部不要做最低点，避免积水直接泡板
- 电池固定要能承受急转和碰撞，避免移动导致重心变化

## 下水前的机械检查表

1. 螺旋桨方向和电机极性一致。
2. 舵机回中后舵面处于机械中心。
3. 联轴器旋转无明显偏摆。
4. 船体静置时左右吃水基本对称。
5. 所有穿线孔和舵杆孔已做密封。
