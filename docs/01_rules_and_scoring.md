# 01 规则与验收

## 说明

仓库里目前没有正式比赛规则原文，所以本文件先作为“项目验收表”和“比赛规则占位页”使用。拿到正式规则后，建议把原文链接、关键约束和评分细则补到本文件顶部。

## 当前建议的功能验收项

| 项目 | 验收标准 | 当前状态 |
| --- | --- | --- |
| 遥控接收 | 能稳定识别 NEC 红外帧和重复帧 | 已实现 |
| 前进/后退档 | `*` 进前进档，`#` 进倒档 | 已实现 |
| 油门控制 | `UP/DOWN` 按步进调速，带平滑输出 | 已实现 |
| 舵机控制 | `LEFT/RIGHT` 调整舵角，`OK` 回中 | 已实现 |
| 停车/复位 | `0` 清零油门并回中舵角 | 已实现 |
| 失联保护 | 一定时间无有效遥控指令后切零油门 | 已实现 |
| 低压保护 | 低压输入触发后切零油门 | 接口已预留 |
| 急停 | 外部输入可直接触发急停 | 接口已预留 |

## 当前建议的工程评分项

| 维度 | 建议权重 | 说明 |
| --- | --- | --- |
| 基础功能完成度 | 40% | 能否稳定完成前进、后退、转向、停车 |
| 安全性 | 20% | 是否具备失联、急停、低压等保护 |
| 结构设计 | 15% | 机械布置、防水、布线、维护性 |
| 文档完整度 | 15% | 接线图、BOM、测试记录是否齐全 |
| 调试与复现性 | 10% | 代码结构、参数管理、测试闭环是否清晰 |

## 正式规则补充时建议关注

- 船体尺寸和质量限制
- 动力和电池规格限制
- 遥控方式是否有限制
- 自动 / 半自动功能是否允许
- 评分是计时、定点、避障还是综合表现
- 是否要求现场答辩、图纸或测试记录
